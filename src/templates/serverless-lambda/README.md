
# Serverless Lambda Template


## Install
```sh
yarn
```

## Run
```sh
yarn start:dev
```

## List all routes created
```sh
yarn routes
```

## Folder Structure (/root)
```
.
├── helpers
└── src
    ├── infra
    │   ├── config
    │   └── container
    ├── modules
    │   └── RandomSample
    │       └── Sample
    │           ├── interfaces
    │           └── useCases
    ├── server
    ├── shared
    │   ├── adapters
    │   ├── contracts
    │   ├── decorators
    │   ├── exceptions
    │   ├── responses
    │   └── types
    └── utils
```
