
# Typescript Basic Project template


## Install
```sh
yarn
```

## Run
```sh
yarn start
```
## Folder Structure 

```sh
.
├── package.json
├── README.md
├── src
│   ├── index.ts
│   ├── interfaces
│   │   └── sample.interface.ts
│   ├── services
│   │   └── sample.service.ts
│   └── shared
│       └── Logger.ts
├── tsconfig.json
├── tslint.json
└── yarn.lock

```
